# Atomic Reactor CLI


```
Usage: arcli [command] [options]

Options:
  -v, --version                 output the version number
  -h, --help                    output usage information

Commands:
  config [options]              Set config.json key, value pairs
  commander [options]           Create a CLI function

  actinium <install> [options]  Install Actinium
  cloud [options]               Creates a new Parse.Cloud function

  reactium <install> [options]  Install Reactium
  reactium <update> [options]   Update Reactium
  component [options]           Creates or replaces a Reactium Component
  rename [options]              Rename a component
  style [options]               Create a style sheet
  element [action] [options]    Manage toolkit elements
```
